## 🚀 Getting Started

### Prerequisites
- Python 3.9 or higher
- pip (Python package manager)
- Git

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/maniker18/generative-ai-roadmap.git
cd generative-ai-roadmap
```

2. **Create a virtual environment:**
```bash
python -m venv venv

# On Windows:
venv\Scripts\activate

# On macOS/Linux:
source venv/bin/activate
```

3. **Install dependencies:**
```bash
pip install -r requirements.txt
```

### Running Examples

Run any Python file to practice:

```bash
# Run Day 1 exercises
python day_01.py/hello.py
python day_01.py/functions.py
python day_01.py/day_2.py/lists.py

# Run API tests
python day_02.py/api_test.py

# Run agent examples
python day_03.py/agent.py
```
